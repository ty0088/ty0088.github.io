Future Projects

Below are some ideas I have for possible future projects.

This would be a project to make a simple clone of the Discord platform. A platform where users can message each other using multiple media types. The aim would be to incorporate the following features which would be novel for me.

  • Ability to create/join different groups and "chat rooms".
  • Groups/chat rooms can be public/private/invite only.
  • Messages can be in the form of voice/text/images.
  • Real time notifications of messages.

This would be a project to create a travel aggregator web app that allows a user to record trips taken and trips that they would want to take. The aim would be to incorporate the following features as a minimum.

  • Create travel itineraries/plans with locations, things to do, notes.
  • Convert travel itineraries to Past Trips showing places visited/things done.
  • Attach images to past trips.
  • Use Google Maps as the map to show locations/itineraries.
  • Itineraries/trips can be public or private.
